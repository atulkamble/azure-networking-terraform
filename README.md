# Azure Networking Terraform Lab

A complete Terraform setup for Azure Virtual Network, Subnets, NSG, ASG, VNet Peering, VPN (Site-to-Site & Point-to-Site), and ExpressRoute.

## Deploy

```bash
terraform init
terraform plan
terraform apply
````

## Requirements

* Terraform v1.6+
* Azure CLI authenticated session
